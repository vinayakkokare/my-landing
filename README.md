## Vercel v0 Usage (Short Notes)
I used Vercel v0 to accelerate UI generation during development:
- Generated the initial Hero and Features section layouts using v0 component templates.
- Tested multiple CTA/button styles and spacing presets using v0 visual previews.
- Exported the v0-generated markup and converted it into Tailwind-based components.
- Used v0 to quickly iterate layout variations before finalizing responsive behavior.
